## Setup
| Dependency     | Version     |
| -------------- | ----------- |
| Ruby           | **3.4.4**   |
| Sqlite3        | **3.43.2**  |

## Running

Install dependencies:
```
bundle install
```
Run migrations:
```
rails db:migrate
```
Start the application:
```
rails server
```
Run tests:
```
rspec
```
Run linter:
```
standardrb

standardrb --fix
```
